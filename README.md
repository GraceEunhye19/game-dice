Dice Roller App

The Dice roller is an Android application built as an extended version of the project provided in Android Developers Jetpack Composes course.
It provides the freedom to chose the number of dice you want to roll and displays a histroy of all faces rolled to track during any game's process.

---

Features
- The choice to roll one or two dice
- History tracking after each roll

---

Tech Stack
- Language: Kotlin  
- Framework: Android Jetpack Compose  
- Architecture: MVVM  
- Tools: Android Studio, Lottie Animations
